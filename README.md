# Avaliação de Acessibilidade – IFVEST

Este repositório contém a documentação das avaliações de
acessibilidade realizadas nas principais funcionalidades
da plataforma IFVEST, utilizando ferramentas de verificação
automatizada assistida, como WAVE e Accessibility Insights for Web.

As análises incluem evidências visuais, relatórios gerados
automaticamente e sugestões de correção conforme as diretrizes WCAG.
